# 🍽️ Restaurant Booking System

Современное веб-приложение для управления рестораном и онлайн-бронирования столиков.  
Разработано на Java (backend) и React (frontend).

---

## 📌 О проекте

Данное приложение позволяет пользователям удобно бронировать столики в ресторане через веб-интерфейс или API.

### 👤 Возможности пользователя:
- 📅 Бронирование столиков
- 📋 Просмотр доступного времени
- ❌ Отмена бронирования

### 🛠️ Возможности администратора:
- Управление бронированиями
- Контроль загруженности ресторана
- Настройка доступности столиков

---
